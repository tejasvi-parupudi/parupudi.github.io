# parupudi.github.io
Website of Prof Tejasvi Parupudi from Arizona State University

This repository hosts the source for my GitHub Pages portfolio. The site summarizes my research, projects, and contact information. Pages included:

- Home (`index.md`)
- About (`about.md`)
- Projects (`projects.md`)
- Research (`research.md`)
- Contact (`contact.md`)

Built with GitHub Pages / Jekyll. To run locally, use `bundle exec jekyll serve` (or your chosen static site generator / theme).
